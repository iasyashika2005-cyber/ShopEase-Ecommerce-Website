# ShopEase-Ecommerce-Website

A responsive e-commerce product page built to practice JavaScript 
fundamentals — variables, control flow, functions, and DOM interaction.

## Features
- Product-listing page with images, names, and prices in a card layout
- "Add to Cart" functionality using JavaScript
- Dynamic cart total and item count updates
- Responsive layout across mobile, tablet, and desktop

## Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
